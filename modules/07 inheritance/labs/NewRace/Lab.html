Part 1: Animating the turtle and the hare

In this part of the lab we're going to redo the turtle and hare race so you don't have to continually click the mouse to run the race. We will use a simple inheritance hierarchy to accomplish this. At the top of our hierarchy, we will have a class, Racer. This class will implement the functionality required for a racer to run the race autonomously once they are started. The TurtleRacer and HareRacer classes will be sub-classes of the Racer class, which means that they will inherit this functionality. The NewRace class will create the racer objects, put them on the race course, and start them. Each racer will time themselves (fairly!) and report their times.

Make two new Java files TurtleRacer.java and HareRacer.java. Modify the files TurtleRacer.java and HareRacer.java so they implement the TurtleRacer and HareRacer classes, respectively; use your code from Lab 4 to make your classes draw the two animals. Each of these classes should be a sub-class of Racer, which is itself a sub-class of GCompound. This is helpful because it means that the TurtleRacer and HareRacer classes are also sub-classes of the GCompound class. Complete the run() method in the Racer class to give your racers the ability to autonomously complete the race and report their elapsed time.

Part 2: Adding another racer

Make a class for a new animal (not a turtle or hare). Modify the NewRace class so that there are three racers, including one of the new animal.
