Lab 11: Manipulating Text—Pig Latin

Perhaps you remember trying to speak in “Pig Latin” as a child. Words starting with consonants (all of the consonants before the first vowel) have the starting consonants removed from the front of a word and added to the end of the word followed by “ay” so that, for example, “way” becomes “ayway”, “java” becomes “avajay”, “theme” becomes “emethay", and so on. Words starting with a vowel stay the same with a “way” appended to their end so that “elated” becomes “elatedway”, “after” becomes “afterway” etc. In particular, if you had “awesome Pig Latin” it would become “awesomeway igPay atinLay”. If you need to refresh your memory more on Pig Latin, check out this website: English to Pig Latin Translator.

Part 1: A method that translates one word into Pig Latin

You will be completing a program that can take any string of zero or more English words and translate it to a string of equivalent words in Pig Latin. You have been given a complete main program, GraphicsPigLatin, that provides the user interface for the translater and a class, PigLatinString, that represents a string that can be translated from English to Pig Latin. The user interface uses Java GUI components that we haven't studied yet, but you won't need to work with those parts of the program. The translate() method in the PigLatinString class, completed for you, is responsible for translating a PigLatinString object from English to Pig Latin, but it relies on the unfinished translateWord() method. Your job is to complete the translateWord() method in the PigLatinString class. The rules for how the method should work are in the class comment at the top of the file.

Show your work to the TA or lab instructor before you go on.

Part 2: Correcting the capitalization

The translateWord() method doesn't work correctly for words beginning with a capital consonant. Modify this method so that it handles this case correctly; for example, “Pig Latin” will become “Igpay Atinlay” instead of “igPay atinLay.” The Character and String classes should provide methods appropriate for accomplishing this task.

Show your work to the TA or lab instructor before you go on.

Part 3: Translating text files into Pig Latin

Write a new program that opens a Scanner on a text file, reads and translates each line of the file to Pig Latin, and writes each translated line to another file using a PrintWriter. The paths for the files should be provided as command-line arguments, i.e., the args array in the main() method. Study the actionPerformed() method in the GraphicsPigLatin class for example usage of the PigLatinString class and its translate() method.

You can use the two text files provided in the lab directory, or create some of your own, to check your solution for correctness.

Show your completed program to the TA or lab instructor.
